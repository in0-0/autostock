# Product Philosophy

AutoStock is not a black-box trading bot. It is a conservative weekly decision
support system for Korean equities.

## Principles

- Guidance over impulse: produce reviewable recommendations, not hidden actions.
- Explainability over cleverness: every candidate should have filter-level reasons.
- Safety over coverage: partial data success should block risky new buys.
- Local durability: important outputs should survive process crashes and reruns.
- Provider independence: broker and market data integrations should be replaceable.
