# Architect Role

Use this role for connector design, market data fallback strategy, scheduling,
or changes that affect multiple modules.

## Deliverables

- Problem framing and constraints.
- Options with tradeoffs.
- Recommended direction.
- Migration or rollout plan.
- Risks and tests required.

Keep proposals grounded in the existing local-first batch architecture.
